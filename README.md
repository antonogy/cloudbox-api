# Study Project “CloudBox API”

## Description

Study Project “CloudBox API” - an API for storing files in a cloud like Goolgle Drive

## Technical spec
[Spec.md](technical-spec.md)

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
